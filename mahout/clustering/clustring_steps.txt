Mahout provides command line arguments for performing machine learning analysis. In order to perform K-means
clustering following steps are required to be followed:

Step 1: Converting the input text to sequences
		The first step is to convert the input documents into a sequence form using seqdirectory.

./bin/mahout seqdirectory -i ~/Desktop/netflix/cluster_set -o ~/Desktop/netflix/kmeans/parsedtext-seqdir -c UTF-8 -chunk 5

Step 2: Converting the generated sequences to sparse vectors using seq2sparse.
		This step is necessary to obtain vectors from sequence file to use it as part of the input to the clustering process.

 ./bin/mahout seq2sparse -i ~/Desktop/netflix/kmeans/parsedtext-seqdir -o ~/Desktop/netflix/kmeans/parsedtext-seqdir-sparse-kmeans --maxDFPercent 85 --namedVector

Step 3: Running the KMeans clustering

 ./bin/mahout kmeans -i  ~/Desktop/netflix/kmeans/parsedtext-seqdir-sparse-kmeans/tfidf-vectors/ -c ~/Desktop/netflix/kmeans/parsedtext-kmeans-clusters
 	      -o ~/Desktop/netflix/kmeans/parsedtext-kmeans -dm org.apache.mahout.common.distance.CosineDistanceMeasure -x 10 -k 20 -ow --clustering -cl

Step 4: Dumping the results to file
		For viewing the clustering results, it needs to be dumped through the use of a special program called clusterdump. We specify output of the dump with -o.

 ./bin/mahout clusterdump -i ~/Desktop/netflix/kmeans/parsedtext-kmeans/clusters-*-final -d ~/Desktop/netflix/kmeans/parsedtext-seqdir-sparse-kmeans/dictionary.file-0
 	      -dt sequencefile -b 100 -n 20 --evaluate -dm org.apache.mahout.common.distance.CosineDistanceMeasure --pointsDir ~/Desktop/netflix/kmeans/parsedtext-kmeans/clusteredPoints
 	      -o ~/Desktop/netflix/kmeans/cluster-output.txt

Step 5: Dumping the Document to Cluster mapping
		The cluster mapping can be generated by using the seqdumper program. The ID of the cluster is specified by "Key: <clusterid>" in the file and the filename is specified with "value: <documentid>"

 ./bin/mahout seqdumper -i ~/Desktop/netflix/kmeans/parsedtext-kmeans/clusteredPoints/part-m-00000 > ~/Desktop/netflix/kmeans/cluster-points.txt
